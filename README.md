# git-branch-list

List and select a git branch easily.

## Getting Started
```
go install github.com/schoolboybru/git-branch-list
```

## MakeFile

run all make commands with clean tests
```bash
make all build
```
build the application
```bash
make build
```
run the application
```bash
make run
```
Create DB container
```bash
make docker-run
```
clean up binary from the last build
```bash
make clean
```
